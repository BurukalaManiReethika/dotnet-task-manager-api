# 🚀 Task Manager API

A simple REST API built with ASP.NET Core Web API and SQLite.

## Features

- Create Tasks
- Get All Tasks
- Get Task By ID
- Update Tasks
- Delete Tasks
- Swagger Documentation

## Tech Stack

- ASP.NET Core 8
- Entity Framework Core
- SQLite
- Swagger

## Endpoints

| Method | Endpoint |
|---------|----------|
| GET | /api/tasks |
| GET | /api/tasks/{id} |
| POST | /api/tasks |
| PUT | /api/tasks/{id} |
| DELETE | /api/tasks/{id} |
